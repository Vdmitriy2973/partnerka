Общая информация

Что такое партнёрская программа LinkOffer?

LinkOffer — это платформа для управления партнёрским маркетингом, которая соединяет рекламодателей и партнёров. Мы предоставляем инструменты для отслеживания кликов, конверсий и выплат.

Как работает партнёрский маркетинг?

Партнёрский маркетинг — это модель, в которой рекламодатели платят партнёрам за совершение целевых действий. Партнёры размещают рекламные материалы, а система отслеживает эффективность каждого партнёра.

Для партнёров

Как подключиться к проекту в качестве партнёра?
  1. Зарегистрируйтесь в партнёрской программе как партнёр
  2. В разделе Предложения выберите подходящее предложение
  3. Подключитесь к проекту рекламодателя
  4. После подключения к проекту рекламодателя получите партнёрскую ссылку в разделе Проекты
  5. Размещайте ссылку на своих площадках и начинайте зарабатывать
Как получить партнёрскую ссылку?

После подключения к проекту рекламодателя, в личном кабинете в разделе Проекты вы найдёте подключенный проект. Эту ссылку необходимо сгенерировать, заполнив данные и использовать её для продвижения товара или услуги.

Как отслеживать статистику и выплаты?

В личном кабинете доступен подробный отчет по каждому проекту:

  • Количество кликов
  • Количество конверсий
  • CR (Conversion Rate - Конверсионная ставка)
  • Общий заработок

Для рекламодателей

Как подключить проект к партнёрской программе?
  1. Зарегистрируйтесь в партнёрской программе как рекламодатель
  2. Заполните юр. данные в личном кабинете в разделе Реквизиты
  3. В разделе Проекты создайте новый проект
  4. Заполните информацию о проекте: название, URL, описание, шаблон партнёрской ссылки, параметры (опционально), цена за целевое действие
  5. Опубликуйте проект, получите одобрение от модераторов
  6. Сотрудничайте с партнёрами
Как настроить отслеживание кликов и конверсий?

Отслеживание кликов и конверсий подробно описано на странице API документация

Как привлекать качественных партнёров?

Для привлечения лучших партнёров рекомендуем:

  • Создать подробное название проекта
  • Создать подробное описание проекта
  • Установить конкурентные ставки вознаграждения

Как подключить площадку партнёра?

Для подключения вашей площадки:

  1. В личном кабинете перейдите в раздел Мои площадки
  2. Нажмите Добавить площадку
  3. Заполните информацию о площадке: название, тип площадки, описание, URL
  4. После модерации площадка будет активирована

Важно:

Указание точной информации о площадке увеличивает шансы на одобрение заявок рекламодателями.
